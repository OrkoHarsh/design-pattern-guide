# Design Pattern Guide

An immersive, page-by-page learning guide to all **23 Gang of Four** design patterns.

**Live demo:** [design-pattern-guide.netlify.app](https://design-pattern-guide.netlify.app)

---

## Why this exists

Most pattern resources are UML dumps and definitions.  
This guide focuses on **judgment** — when to use a pattern, how it works, and how it looks in code.

One lesson. One page. One clear mental model.

---

## What’s inside

- **23 GoF patterns** — Creational, Structural, Behavioral (including Interpreter)
- **Foundations** — what patterns are, anatomy, history, criticism, classification
- **Story + structure animations** — roles first, then the design structure
- **5 languages per pattern** — Pseudocode, Python, C++, Java, JavaScript
- **Find Your Pattern** — decision path for real design choices
- **Compare Patterns** — side-by-side intent, structure, use / avoid
- **Master Quiz** — interview-style practice
- **Dark / light themes** + mobile-friendly layout
